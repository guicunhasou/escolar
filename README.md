# Site Escolar — Case de Portfólio

Projeto front-end responsivo desenvolvido como case de portfólio, inspirado em uma demanda real no contexto educacional.

A proposta simula um site institucional para uma escola municipal, reunindo informações importantes para a comunidade escolar, como apresentação da escola, comunicados, calendário, blog, contato e visualização de indicadores.

Por questões de privacidade e segurança, esta versão utiliza dados fictícios, identidade visual adaptada e conteúdos anonimizados.

## Objetivo

Criar uma interface simples, organizada e acessível para centralizar informações escolares em um ambiente digital de fácil navegação.

O projeto busca melhorar a comunicação entre escola, estudantes, responsáveis e comunidade, apresentando conteúdos institucionais de forma clara, responsiva e visualmente consistente.

## Funcionalidades

- Página inicial com apresentação institucional
- Área sobre a escola
- Página de comunicados
- Calendário escolar
- Blog/notícias
- Página de contato
- Área planejada para visualização de indicadores escolares simulados
- Layout responsivo para desktop e mobile

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Design responsivo
- Organização modular de arquivos

## Privacidade e anonimização

Este repositório não expõe dados reais da instituição original.

Foram alterados ou substituídos:

- Nome da escola
- Identidade visual institucional
- Endereço
- Telefone
- E-mail
- Dados de indicadores
- Imagens e elementos visuais vinculados ao projeto original

A versão publicada tem finalidade exclusivamente demonstrativa e foi adaptada para apresentação em portfólio.
